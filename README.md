# CST4714 Midterm Project
## Jahmid Ahmed
## Course: CST4714 Database Administration

## Project Title
Campus Event Registration System

## Project Description
This project builds a small PostgreSQL database for managing students, campus events, and event registrations. It demonstrates schema design, data integrity, SQL queries, transactions, concurrency, and admin/operations tasks using Supabase and PostgreSQL.

## Run Order
1. Run `schema.sql`
2. Run `seed_data.sql`
3. Run `queries.sql`
4. Run `admin_ops.sql`

## Files
- `schema.sql` - Creates tables, constraints, and a view
- `seed_data.sql` - Inserts sample data
- `queries.sql` - Required SQL queries
- `admin_ops.sql` - Transaction demos, blocking demo, and admin/ops controls
- `report.md` - Short report for the project
- `er_diagram/er_diagram.pdf` - ER diagram
- `screenshots/` - Evidence screenshots

## Notes
- This project was tested in Supabase SQL Editor.
- Two SQL sessions/tabs are needed for the blocking demonstration.
